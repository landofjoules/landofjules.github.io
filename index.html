ABET Assistant

Demo

During my final year at university, I was recruited to design a web application to be used by the entire Electrical Engineering and Computer Science department. The goal of this project was to simplify a tedious and unorganized accreditation process each professor had to complete using excel files, which had to be managed by my client. I created an interface for professors to enter their data in a manageable way, so that our client could sort through this data using a relational database, which he much preferred. The front end was made with Bootstrap and jQuery, and the backend was built using the Django web framework. The tool was considered a great success and was moved into production in December 2016.

A Photography Portfolio

Demo

The Game of Life

Visit

John Conway's Game of Life is a simple computational experiment with fascinating results, and a devoted community. However, the leading web-playgrounds available are unpleasant to look at and difficult to use. I built gameoflife.xyz using HTML5 Canvas to provide a better place to experiment with this complex system, and to understand the set of rules that can be used to create a universal Turing machine. It is intended for demonstrative and causal purposes, and has a number of features that assist with explanation and experimentation.
