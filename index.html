Team Leader / UI Developer

During my final year at university, I was selected to lead a four-person senior design team to create the ABET Assistant, a user interface designed to assist the professor responsible for overseeing the collection of accreditation data across the Electrical Engineering and Computer Science department. I was also responsible for the design of the entire front end. The goal of this project was to simplify a tedious and unorganized accreditation process each professor had to complete using excel files, which had to be managed by our client. We created an interface for professors to enter their data in a manageable way, so that our client could sort through this data using a relational database, which he much preferred. The front end was made with Bootstrap and jQuery, and the backend was built using the Django web framework. The tool was considered a great success and was moved into production in December 2016.

The Game of Life

John Conway’s Game of Life is a simple computational experiment with fascinating results, and a devoted community. However, the leading web-playgrounds available are unpleasant to look at and difficult to use. I built gameoflife.xyz using HTML5 Canvas to provide a better place to experiment with this complex system, and to understand the set of rules that can be used to create a universal Turing machine. It is intended for demonstrative and causal purposes, and has a number of features that assist with explanation and experimentation.
